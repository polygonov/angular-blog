Этот проект является моделью полноценного веб-приложения (PWA) и нуждается в развитии и наполнении. Он создан только для демонстрации работы с фреймворком Angular 8. Проект включает в себя два модуля (основной и административный), некоторое количество компонентов и сервисов. 

Основной модуль - это область отображения блога, он является публичным и любой пользователь сети может получить доступ к чтению информации (блога) по прямой url ссылке. Доступен предпросмотр всех постов пользователя, а также просмотр каждого поста в отдельности.

Модуль администратора - это защищенная область. Пользователю необходимо зарегистрироваться, чтобы попасть в эту часть приложения и создать собственный блог. В дальнейшем, чтобы вновь получить доступ к редактированию своего контента, пользователю необходимо будет авторизоваться. Модуль администратора содержит в себе компонент регистрации, компонент авторизации, страницу создания поста, страницу с предпросмотром постов для выбора к редактированию, и страницу редактирования.

Данный проект содержит в себе следующий функционал Angular-фреймворка: модульная архитектура приложения (SPA), REST-взаимодействие с Firebase (серверное приложение, NoSQL база данных), авторизация через API Firebase Auth, валидация компонентов, реактивные формы, защита роутов, реализован принцип lazy loading (ленивая загрузка), работа с плагинами.
